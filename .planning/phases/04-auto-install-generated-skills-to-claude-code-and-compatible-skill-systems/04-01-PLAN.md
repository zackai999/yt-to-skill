    2. Create `tests/test_installer.py` with all test cases above. Tests must use `tmp_path` fixture to create fake agent skill directories and fake SKILL.md files with YAML frontmatter. Monkeypatch `Path.home()` to point to tmp_path for agent detection tests.

    3. Create `yt_to_skill/installer.py` implementing:

    **Constants (evaluate at call time, not import time per anti-pattern from RESEARCH.md):**
    - `AGENT_GLOBAL_PATHS` dict: claude-code -> ~/.claude/skills/, codex -> ~/.agents/skills/, cursor -> ~/.cursor/skills/, gemini -> ~/.gemini/skills/, copilot -> ~/.copilot/skills/
    - `AGENT_PROJECT_PATHS` dict: claude-code -> .claude/skills/, codex -> .agents/skills/, cursor -> .cursor/skills/, gemini -> .gemini/skills/, copilot -> .github/skills/ (note copilot uses .github/)
    - Make these functions returning dicts (not module-level constants) so Path.home() is evaluated at call time, enabling test monkeypatching.

    **Functions:**
    - `get_global_paths() -> dict[str, Path]`: Returns AGENT_GLOBAL_PATHS with Path.home() evaluated at call time
    - `get_project_paths() -> dict[str, Path]`: Returns AGENT_PROJECT_PATHS (relative, no home)
    - `detect_installed_agents() -> list[str]`: Check which agents have global skills directory present. Skip silently + debug log when not found (loguru).
    - `sanitize_skill_name(name: str) -> str`: Lowercase, replace non-alphanumeric with hyphens, collapse consecutive hyphens, strip leading/trailing hyphens, truncate to 64 chars. Use `re.sub(r'[^a-z0-9-]', '-', name.lower())` then collapse + strip.
    - `_inject_provenance(skill_md_text: str, video_id: str) -> str`: Parse frontmatter between --- markers, add source_video_id and installed_at (UTC ISO), re-serialize with yaml.dump. Return unchanged if no frontmatter.
    - `_check_conflict(install_path: Path, skill_name: str) -> bool`: Return True if (install_path / skill_name) exists.
    - `install_skill(src_dir: Path, target_base: Path, skill_name: str, video_id: str, *, overwrite: bool = False) -> Path`: Create parent dirs, check conflict (raise if exists and overwrite=False), rmtree existing if overwrite=True, copytree, inject provenance into the installed SKILL.md. Return installed path.
    - `list_installed_skills() -> list[dict]`: Scan all global and project paths. For each SKILL.md, parse frontmatter. Include only those with source_video_id. Return list of dicts with keys: agent, name, source_video_id, installed_at, path, scope (global/project).
    - `uninstall_skill(name: str) -> list[Path]`: Scan all global and project paths for name/ directory containing SKILL.md with source_video_id. Remove with shutil.rmtree. Return list of removed paths.

    4. Run tests to confirm all pass.
