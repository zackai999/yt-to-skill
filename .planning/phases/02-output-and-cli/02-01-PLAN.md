    **Step 1: Create `yt_to_skill/errors.py`**

    Define `SkillError(Exception)` base class with `category: str` and `suggestion: str` class attributes. Override `__init__(self, message: str)` to format as `f"[{self.category}] {message} — {self.suggestion}"`. Create four subclasses:
    - `NetworkError` — category="NETWORK", suggestion="Check internet connection or retry later."
    - `ExtractionError` — category="EXTRACTION", suggestion="Video may lack captions; delete work/<video_id>/ and rerun."
    - `LLMError` — category="LLM", suggestion="Check OPENROUTER_API_KEY and model availability on openrouter.ai."
    - `FormatError` — category="FORMAT", suggestion="extracted_logic.json may be malformed; delete work/<video_id>/ and rerun."

    **Step 2: Create `yt_to_skill/stages/skill.py`**

    Two public functions:

    `render_skill_md(extraction: TradingLogicExtraction) -> str`:
    - Build YAML frontmatter: `name` = `extraction.video_id.lower()[:64]`, `description` = summary of strategy names (max 1024 chars), `metadata` block with `version: "1.0"`, `source_url`, `source_language`
    - Build body with four top-level sections per the locked decision. If multiple strategies, repeat the four sections per strategy with strategy name as a prefix/heading
    - REQUIRES_SPECIFICATION: For each item in `strategy.unspecified_params`, render as `> **REQUIRES_SPECIFICATION: {param}** — exact value not stated in video` inline within the section that contains that param (entry_criteria params go under Entry/Exit, exit_criteria params go under Entry/Exit, risk-related under Risk Management)
    - Empty strategies: Write a "No trading strategies were extracted from this video." message
    - EntryCondition rendering: bullet format — `- **{indicator}**: {condition}` with optional `(value: {value})`, `(timeframe: {timeframe})`, `(confirmation: {confirmation})`. Also include `raw_text` as italic fallback if structured fields are sparse

    `run_skill(video_id: str, work_dir: Path, skills_dir: Path, *, force: bool = False) -> StageResult`:
    - `skill_path = skills_dir / video_id / "SKILL.md"`
    - If not force and `artifact_guard(skill_path)` returns True: return StageResult with skipped=True
    - Load extraction via `TradingLogicExtraction.from_file(work_dir / video_id / "extracted_logic.json")`. Wrap in try/except; on FileNotFoundError or pydantic ValidationError, raise `FormatError`
    - Create directory scaffold: `skills_dir / video_id`, then `assets/`, `scripts/`, `references/` subdirs
    - Call `render_skill_md(extraction)`, write to skill_path
    - Return StageResult(stage_name="skill", artifact_path=skill_path, skipped=False)
