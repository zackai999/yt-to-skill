       `deduplicate_frames(frame_paths: list[Path], threshold: int = 10) -> list[Path]`: Iterate sorted frame_paths, compute `imagehash.phash(Image.open(path))` for each, keep frame if hamming distance to ALL kept hashes > threshold. Return kept list.

       `run_keyframes(video_id: str, work_dir: Path, config: PipelineConfig) -> StageResult`:
       - Sentinel guard: `keyframes_done = work_dir / video_id / "keyframes.done"` -- if exists, return StageResult(skipped=True)

       - `manager.detect_scenes(video, show_progress=False)`
       - `scene_list = manager.get_scene_list()`

       - Use `save_images(scene_list, video, num_images=1, image_extension='png', encoder_param=9, output_dir=str(output_dir), show_progress=False)` -- NOTE: scene_list first arg, then video

       - Run `deduplicate_frames()` on the renamed PNGs, delete non-kept files

       - Write sentinel: `keyframes_done.write_text(str(len(kept_frames)))`
