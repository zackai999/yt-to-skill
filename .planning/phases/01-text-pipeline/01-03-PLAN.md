  <name>Task 2: Transcript stage — captions + Whisper fallback with quality heuristics</name>
  <files>yt_to_skill/stages/transcript.py, tests/test_transcript.py</files>
  <behavior>
    - Test: fetch_captions returns (segments, language_code) for a video with available captions
    - Test: fetch_captions returns None when TranscriptsDisabled is raised
    - Test: fetch_captions returns None when NoTranscriptAvailable is raised
    - Test: fetch_captions prioritizes zh/zh-Hans/zh-Hant languages, then falls back to any
    - Test: is_caption_quality_acceptable returns False when Music tag ratio > 0.3
    - Test: is_caption_quality_acceptable returns False when total chars < duration * 2
    - Test: is_caption_quality_acceptable returns False when short segment ratio > 0.6
    - Test: is_caption_quality_acceptable returns True for normal quality segments
    - Test: run_transcript tries captions first, falls back to Whisper on failure
    - Test: run_transcript falls back to Whisper when caption quality is poor
    - Test: Whisper transcription uses vad_filter=True
    - Test: Whisper model is loaded with Belle-whisper-large-v3-zh-punct model ID
    - Test: run_transcript writes TranscriptArtifact to raw_transcript.json
    - Test: run_transcript skips when raw_transcript.json already exists
  </behavior>
  <action>
    1. Create yt_to_skill/stages/transcript.py with:

       - fetch_captions(video_id: str, language_priority: list[str] = None) -> tuple[list[dict], str] | None:
         Default language_priority: ["zh", "zh-Hans", "zh-Hant", "en"].
         Try YouTubeTranscriptApi.list_transcripts(video_id).
         For each language in priority, try find_transcript([lang]).fetch().
         If no priority match, take any available transcript.
         Convert FetchedTranscriptSnippet to dicts: {start, end, text}.
         Catch TranscriptsDisabled, NoTranscriptAvailable, VideoUnavailable -> return None.

       - is_caption_quality_acceptable(segments: list[dict], video_duration_s: float) -> bool:
         Calculate total_chars, music_tag_ratio, short_segment_ratio (segments with <3 words).
         Return False if: total_chars < duration * 2, music_tag_ratio > 0.3, short_segment_ratio > 0.6.
         Return True otherwise.

       - _whisper_model global variable (lazy singleton).
       - get_whisper_model(device: str = "cpu", compute_type: str = "int8") -> WhisperModel:
         Lazy-load Belle-whisper: "Huan69/Belle-whisper-large-v3-zh-punct-fasterwhisper".

       - transcribe_audio(audio_path: Path, device: str = "cpu", compute_type: str = "int8") -> list[dict]:
         Call model.transcribe with language="zh", task="transcribe", vad_filter=True, beam_size=5.
         Return list of {start, end, text} dicts.

       - run_transcript(video_id: str, work_dir: Path, config: PipelineConfig) -> StageResult:
         Artifact guard: if raw_transcript.json exists, skip.
         Load metadata.json for video_duration_s.
         Try fetch_captions. If result is not None, check quality with is_caption_quality_acceptable.
         If captions good: method="captions", caption_quality="good".
         If captions exist but poor quality: log warning, proceed to Whisper, caption_quality="poor".
         If no captions: caption_quality="missing", proceed to Whisper.
         Whisper path: call download_audio from ingest module, then transcribe_audio.
         Build TranscriptArtifact, serialize to raw_transcript.json.
         Return StageResult.

    2. Create tests/test_transcript.py:
       - Mock youtube_transcript_api classes for caption tests.
       - Mock faster_whisper.WhisperModel for Whisper tests.
       - Test quality heuristics with synthetic segment data.
       - Test the full run_transcript flow: captions available -> uses captions; captions fail -> calls Whisper.
       - Test artifact guard with pre-existing raw_transcript.json.
  </action>
