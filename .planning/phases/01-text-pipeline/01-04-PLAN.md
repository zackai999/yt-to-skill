    - Test: run_filter skips Stage 2 (LLM call) when metadata_prefilter returns False

    - Test: run_filter returns is_strategy=False with reason when video is filtered out
  </behavior>
  <action>
    1. Create yt_to_skill/stages/filter.py with:

       - STRATEGY_KEYWORDS: set of terms that suggest trading content. Include both English and Chinese: {"strategy", "trading", "entry", "exit", "setup", "indicator", "RSI", "MACD", "support", "resistance", "breakout", "pullback", "策略", "入场", "出场", "止损", "止盈", "指标", "趋势", "突破", "回调", "均线", "布林"}

       - NON_STRATEGY_KEYWORDS: set of terms that suggest non-strategy content: {"vlog", "daily life", "news", "reaction", "unboxing", "review", "opinion", "podcast", "interview", "Q&A", "giveaway"}

       - metadata_prefilter(title: str, description: str, tags: list[str]) -> tuple[bool, str]:
         Score: +1 for each STRATEGY_KEYWORD found in title+description+tags, -1 for each NON_STRATEGY_KEYWORD.
         If score <= 0: return (False, "Metadata pre-filter: no strategy indicators found").
         If score > 0: return (True, "Metadata pre-filter: strategy indicators found").
         Case-insensitive matching. Check both Chinese and English terms.

       - run_filter(video_id: str, work_dir: Path, config: PipelineConfig, llm_client: OpenAI | None = None) -> StageResult:
         Artifact guard: if filter_result.json exists, skip.
         Load metadata.json for title, description, tags.
         Stage 1: metadata_prefilter. If False, write FilterResult(is_strategy=False, transcript_pass=None) and return.
         Stage 2: Load raw_transcript.json, take first 500 words of concatenated text. Call classify_content via llm_client. If llm_client is None, pass Stage 2 (conservative — assume strategy).
         Write FilterResult to filter_result.json. Return StageResult.

    2. Create tests/test_filter.py:
       - Test metadata_prefilter with various title/description combinations.
       - Mock classify_content to test Stage 2 flow.
       - Test artifact guard.
       - Test that Stage 2 is skipped when Stage 1 fails (verify classify_content not called).

       - extract_glossary_additions(response: str) -> list[tuple[str, str]]:
         Parse GLOSSARY_ADDITIONS section from LLM response if present.
         Return list of (chinese_term, english_term) tuples.
         Log each addition via loguru for review.

       - run_translate(video_id: str, work_dir: Path, config: PipelineConfig, llm_client: OpenAI | None = None) -> StageResult:
         Artifact guard: if translated.txt exists, skip.
         Load raw_transcript.json into TranscriptArtifact.
         Detect source language.
         If English: write transcript text directly to translated.txt (no LLM call needed).
         If non-English: load glossary via load_glossary(config glossary path or default path).
           Concatenate transcript segments into full text (preserving timestamps as markers).
           Call translate_text(llm_client, text, glossary, config).
           Extract and log any GLOSSARY_ADDITIONS from response.
           Strip GLOSSARY_ADDITIONS section from the translated output before saving.
           Write clean translated text to translated.txt.
         Return StageResult.
