  <action>
    1. Create yt_to_skill/orchestrator.py with:

       - run_pipeline(video_id: str, config: PipelineConfig) -> list[StageResult]:
         Create work directory: config.work_dir / video_id. Mkdir parents=True, exist_ok=True.

         Create LLM clients once:
           openai_client = make_openai_client(config)
           instructor_client = make_instructor_client(config)

         Run stages in sequence, collecting StageResults:

         1. ingest_result = run_ingest(video_id, config.work_dir, config)
            Log: "Stage: ingest — {skipped|completed}"

         2. transcript_result = run_transcript(video_id, config.work_dir, config)
            Log: "Stage: transcript — {skipped|completed}"

         3. filter_result = run_filter(video_id, config.work_dir, config, llm_client=openai_client)
            Log: "Stage: filter — {skipped|completed}"
            If filter says is_strategy=False: log "Video filtered as non-strategy", return results so far.

         4. translate_result = run_translate(video_id, config.work_dir, config, llm_client=openai_client)
            Log: "Stage: translate — {skipped|completed}"

         5. extract_result = run_extract(video_id, config.work_dir, config, instructor_client=instructor_client)
            Log: "Stage: extract — {skipped|completed}"

         Wrap each stage call in try/except. On error: log the error with stage name and video_id, append StageResult with error field set, continue to return partial results (do NOT abort the entire pipeline for a single stage failure — but downstream stages that depend on the failed stage's artifact will naturally skip or fail).

         Return all StageResults.

       - extract_video_id(url: str) -> str:
         Parse YouTube URL formats: watch?v=ID, youtu.be/ID, shorts/ID.
         Return the video ID string. Raise ValueError for unrecognized URL formats.
